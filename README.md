# Coffee quality classifier
A machine learning project that classifies quality using a neural network built TensorFlow and Keras. The project include data preprocessing, exploratory data analysis, feature engineering, model training and evaluation.
## Objective
The objective of this project is to build a neural network model capable of predicting coffee quality using data from `coffee_quality.csv`. This project was created to practice data preprocessing, feature selection, model training, and evaluation with TensorFlow/Keras.

## Dataset
- **Source:** https://www.kaggle.com/datasets/fatihb/coffee-quality-data-cqi
- **Number of Samples:** 207 
- **Features:** 12
- **Target:** Coffee quality
- **Description**: This dataset contains physicochemical characteristics used to predict coffee quality.
